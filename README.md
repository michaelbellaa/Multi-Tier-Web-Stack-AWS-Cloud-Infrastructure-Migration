# Multi-Tier-Web-Stack-AWS-Cloud-Infrastructure-Migration
## Scalable Infrastructure | Custom VPC | Route 53 Service Discovery | Linux Hardening
This project demonstrates the migration of a 5-tier web application from a local Vagrant environment to a production-ready AWS Cloud Infrastructure. By deploying onto Amazon EC2 within a custom VPC, this architecture achieves physical separation of concerns, enhanced security through "Least Privilege" Security Groups, and internal service discovery via Route 53.
